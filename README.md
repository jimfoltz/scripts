# scripts for Windows cmd.exe

Scripts and shortcuts mostly for working in Windows cmd.exe.

ps.cmd - interactive fzf-based process viewer/killer. 


![screenshot](https://imgur.com/WG9JkUV.png)



Keys:
        F1 - refresh the list and sort by size
        F2 - refresh the list and sort by name
        F5 - kill process by pid
        F6 - kill process by name
        F9 - toggle process info window (provided by wmic)
        F10 - launch taskmgr
        ESC - Quit
