# tg.rb

Ripped from https://github.com/mkaz/termgraph but more amateurish.

tg.rb: make simple graphs on your terminal

Usage:
    
    > generate-some-data | tg

Given this input:
```
2006,1331
2007,1455
2008,1070
2009,989
2010,704
2011,517
2012,337
2013,274
2014,293
2015,370
2016,319
2017,274
2018,294
2019,251
2020,272
```

Produces something like this output:

```
2006: ********************************************************************************************** 1331
2007: ******************************************************************************************************* 1455
2008: **************************************************************************** 1070
2009: ********************************************************************** 989
2010: ************************************************** 704
2011: ************************************* 517
2012: ************************ 337
2013: ******************** 274
2014: ********************* 293
2015: ************************** 370
2016: *********************** 319
2017: ******************** 274
2018: ********************* 294
2019: ****************** 251
2020: ******************** 272
```

```
Usage: tg [options]
    -s [c]                           Field separator defaults to comma
        --record-separator [c]       Record Separator defaults to newline
    -h, --help                       Help
        --title [The Title]
                                     Title
        --[no-]color                 color on
        --ascii                      ascii on
    -c, --column=N                   Column
        --sort                       Sort
    -r, --reverse                    Reverse Sort
    -w, --width=N                    Width

```
